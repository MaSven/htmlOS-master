Hello this instruction will show you how to run htmlOS on linux device.
1.Create /sdcard and /storage/emulated/0 folders in your root(/) catalog
2.Extract htmlOS.zip archive to /sdcard folder
3.Extract htmlOS.zip archive to /storage/emulated/0/
5.Open Google Chrome(or any other wich supports File System API)browser
6.Open this file in browser by typing file:///sdcard/htmlOS/GRUB.html
